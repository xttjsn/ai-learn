# Modern Python Practice

Work through these files in order. Each builds on the previous.

| # | File | Topics |
|---|------|--------|
| 1 | `01_typing_basics.py` | Type hints, Union, Optional, generics |
| 2 | `02_dataclasses.py` | @dataclass, field(), post_init, frozen |
| 3 | `03_pydantic_models.py` | Pydantic BaseModel, validation, serialization |
| 4 | `04_protocols.py` | Structural subtyping, Protocol, runtime_checkable |
| 5 | `05_asyncio_basics.py` | async/await, gather, create_task |
| 6 | `06_asyncio_patterns.py` | Semaphores, queues, timeout, streaming |
| 7 | `07_all_together.py` | Mini project combining everything |

Run each file directly: `python3 01_typing_basics.py`

Each file has exercises marked with `# TODO` — fill them in, then run to check.
